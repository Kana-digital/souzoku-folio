プライバシーポリシー

最終更新日: 2026年4月14日

想続フォリオ（以下「本アプリ」）は、大切な方への備忘録として、ご自身で情報を整理・記録するためのアプリです。本ポリシーは、本アプリにおけるデータの取り扱いについて説明します。

1. 本アプリが扱うデータと保存場所

本アプリに入力された情報（名称、カテゴリ、金額、メモ等）は、お使いの端末内にのみ保存されます。外部のサーバーに送信されることはありません。インターネット接続がなくてもお使いいただけます。

アカウント登録は不要です。生体認証（Face ID / Touch ID 等）による画面ロックは、すべて端末上で処理されます。

無料プランではGoogle AdMobによる広告が表示されます。パーソナライズ広告は無効にしており、追跡型の広告は配信されません。

プレミアムプラン（広告非表示）の購入はApple App Store / Google Play Storeを通じて処理されます。決済情報が運営者に届くことはありません。

2. プレミアムプランについて

プレミアムプラン（¥110/月 または ¥980/年）に加入すると広告が非表示になり、広告関連のデータ送信も停止します。

3. 外部サービスとの連携

本アプリはRevenueCat（課金状態の管理）およびGoogle AdMob（無料プランの広告表示）を利用しています。いずれも機能提供に必要な範囲でのみ利用しており、入力されたデータを販売・共有することはありません。

4. データの削除

アプリを削除（アンインストール）すると、端末内のデータは消去されます。

5. ポリシーの変更

本ポリシーは予告なく変更する場合があります。
